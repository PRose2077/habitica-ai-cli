
# 在 Habitica 中使用 Markdown (Markdown in Habitica)

Habitica 支持使用修改后的 Markdown 语法来格式化任务、个人主页和消息中的文本，以及插入图片。

*注意：Markdown 格式在移动端应用上的显示效果可能会与预期有所不同。*

## 标题 (Headings)

你可以通过在文本前添加 1 到 3 个 `#` 符号来创建标题。使用的 `#` 数量决定了标题的层级和大小。

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式 (Styling Text)

你可以使用粗体、斜体或删除线来强调文本。

| 样式 | 语法 | 示例 | 输出效果 |
| --- | --- | --- | --- |
| **粗体 (Bold)** | `** **` | `**这是粗体文本**` | **这是粗体文本** |
| *斜体 (Italic)* | `* *` 或 `_ _` | `*这段文本是斜体*` | *这段文本是斜体* |
| ~~删除线 (Strikethrough)~~ | `~~ ~~` | `~~这是错误的文本~~` | ~~这是错误的文本~~ |
| **粗体嵌套*斜体*** | `** **` 和 `_ _` | `**这段文本_非常_重要**` | **这段文本*非常*重要** |
| ***全粗斜体*** | `*** ***` | `***这些文本都很重要***` | ***这些文本都很重要*** |

## 引用文本 (Quoting Text)

你可以使用 `>` 来引用文本。

```markdown
> 那么第二顿早餐呢？(What about second breakfast?)
```

## 代码 (Code)

你可以使用单反引号来高亮代码。反引号内的文本将不会被渲染格式化。

```markdown
这是一个 `代码 (code)` 的示例。
```

## 链接 (Links)

你可以通过将链接文本包裹在方括号 `[ ]` 中，然后将 URL 链接包裹在圆括号 `( )` 中来创建文本链接。

```markdown
这是一个 [Habitica](https://habitica.com) 链接。
```

## 图片 (Images)

若要显示图片，请先输入 `!`，将替代文本 (Alt text，即对图片内容的简短描述) 包裹在初括号 `[ ]` 中，最后将图片的 URL 包裹在圆括号 `( )` 中。

```markdown
![具有红色和蓝色尖尾、白腿和黄嘴的皇家紫狮鹫像素画。](https://habitica.com/static/presskit/Logo/Habitica%20Gryphon.png)
```

## 列表 (Lists)

你可以通过在一行或多行文本前加上 `-`、`*` 或 `+` 来创建**无序列表**：

```markdown
- 小火龙 (Charmander)
* 杰尼龟 (Squirtle)
+ 妙蛙种子 (Bulbasaur)
```

要创建**有序列表**，请在每行前面加上数字：

```markdown
1. 水伊布 (Vaporeon)
2. 雷伊布 (Jolteon)
3. 火伊布 (Flareon)
```

## 使用表情符号 (Using Emojis)

你可以通过输入被冒号包裹的表情代码 `:emoji:`（例如 `:smiley:`）来添加表情符号。

```markdown
:smiley:
```
